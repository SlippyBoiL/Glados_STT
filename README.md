# GLaDOS V20.1 (Experimental Core)

> *"I'm different."*

This is an experimental Python-based Home Automation and Assistant Core designed to mimic GLaDOS from Portal. It uses OpenAI-compatible APIs (Perplexity, Local LLMs) for intelligence, local speech recognition for hearing, and Govee APIs for environment control.

**⚠️ WARNING: THIS SOFTWARE IS HIGHLY EXPERIMENTAL.**
It has bugs. It has an attitude. It executes Python code generated by an AI directly on your machine. **Use at your own risk.**

---

## 🧠 AI Model Recommendation

While this system supports cloud models (Perplexity `sonar-pro`, OpenAI `gpt-4o`), **running a Local AI is significantly better.**

* **Latency:** Local models (Llama 3, Mistral) react faster than cloud requests.
* **Personality:** Local models can be uncensored and fine-tuned for the GLaDOS persona, whereas cloud models often refuse to be "mean" or "passive-aggressive" due to safety filters.
* **Privacy:** Keeps your "World Domination Plans" off cloud servers.

## 🔌 Capabilities & Skills

The system dynamically loads skills from the `plugins/` directory.

### 🏠 Home & OS Control
* **App Control:** Open or Close any Windows application (e.g., *"Open Steam"*, *"Kill Chrome"*).
* **Light Control:** Full control over Govee Lights (Brightness, Color, On/Off) via API.
* **Shutdown Protocol:** Can schedule (or abort) a system shutdown.

### 📡 Network Surveillance
* **Public IP Fetcher:** Reports your current Public IP, ISP, and Geolocation.
* **WiFi Key Extractor:** Grabs the saved password for the current WiFi network.
* **Network Scan:** Scans the local ARP table to see connected devices.
* **Port Scanner:** Checks localhost for open common ports (HTTP, SSH, FTP).
* **Latency Test:** Pings Google DNS to check internet health.
* **Site Check:** Checks if a specific website is down or just unreachable for you.

### 🛠 System Diagnostics
* **System Specs:** Detailed readout of CPU, OS, and Python version.
* **Disk Usage:** Reports total, used, and free space on the main drive.
* **Process List:** Snapshots the top 10 running processes.
* **Temp Cleaner:** Deletes files from the Windows `%TEMP%` directory.

### 🔧 Utilities
* **Weather:** Fetches current atmospheric reports.
* **Dictionary:** Definitions for words via free API.
* **Password Gen:** Generates high-entropy secure strings.
* **UUID Gen:** Creates unique identifiers.
* **Crypto Tracker:** Fetches current Bitcoin price.

### 🎲 Fun / Data
* **Coin Flip:** Binary decision maker.
* **Dice Roll:** Supports D4, D6, D20, etc.
* **ISS Tracker:** Reports the real-time coordinates of the Space Station.
* **NASA APOD:** Fetches the Astronomy Picture of the Day.

### 📦 Meta-Skills (Management)
* **Auto Import:** Automatically scans `new_skills.txt` and imports all found code blocks as new skills.
* **Interactive Import:** Asks you for a specific filepath to import skills from.

## 🐛 Known Issues (The "Features")

1.  **Ghost Listening:** The STT (Speech-to-Text) sometimes picks up ambient noise as commands.
2.  **Code Hallucination:** If the AI generates bad Python code, the execution execution handler will catch the error, but the task will fail.
3.  **Process Killing:** The "Kill" command is aggressive. It *will* close the app you ask it to, even if you have unsaved work. It does not care about your data.
